# R2-三三制：同一根因并行三个不同策略家族

R2芯片智能体 · 05/09

并行扇出、切割、重构三家族，统一综合与门禁，按score排序。

| 根因 | 策略（三家族示例） |
|---|---|
| 扇出类 | 寄存器复制 或 局部缓冲 |
| 组合过深 | 切割与分段寄存 或 流水化 或 层次重构 |
| 晚选择 | 选择前移 或 早决策信号生成 或 晚绑定 |
| 长表达式 | 拆中间量 或 公共子表达式 或 结构重写 |

同家族参数变体不计入三路（如缓冲大小、寄存器数目、分段粒度等仅为参数变体）。

## 护栏（必须遵守）
- **目标只有 score 改进**
- **门禁失败不得评分**
- **基线不得移动，失败必须入库**
- 生成可以快，错误不能漏。

## Dark flow 五步
1. **01 诊断**
2. **02 三路取样**
3. **03 门禁计分**
4. **04 复盘沉淀**
5. **05 新问题再 loop**

门禁与 score 能自动则自动；灰区才人工，口径事先写进 GOAL。
