# 4 乘性内核

R2-AIOS

Harness 是卯口。Oracle 缩小决策空间。Memory 让下次起跑线前移。Loop 提供动力。缺一为零。

## 01 Harness — 卯口

不执行，只定义接受形状。编排 agent / parallel / pipeline / while。挖掉的空间比留下的实体更关键。

## 02 Oracle — 决策空间收缩

机械裁判，不是 LLM。每一声「不对」不是否定，是精度——Spike diff · Formal · DRC。失败路径被逐一排除。可能性本身在缩窄：发散 ↺ 回 Agent 自转 → 收敛 ● 全绿 → 人签核。

## 03 Memory — 回弹力的工程复现

木头回弹是天生的——撒力即恢复。Memory 飞轮是造出来的——Trace 捕获 → 蒸馏 → 进化 → 下次起点更高。物质基础不同，因果结构同态：冲击 → 变形 → 恢复 → 更强。真正让碰壁长骨头的，是 Memory。

## 04 Loop — 动力

Intent → Context → Action → Observation → Adjustment。跑到 Oracle 全绿才停。没有 Loop，卯口是空的，榫头不动，键无用。

## 两层合力 = 燕尾效应

**Oracle 实时** 每声「不对」排除一条路径。MAX_ITER −1。决策空间逐轮缩窄。发散 ↺ 自转 → 收敛 ● 签核。

**Memory 跨会话** 失败 trace → 蒸馏 → 下次起点更高 → 更少碰壁。越压越紧的真正来源。物质基础不同，因果结构同态。

发散 ↺ 回 Agent 自转。收敛 ● 全绿 → 人签核。Harness 内 Agent 自主，Harness 外只有人类 taste。
