# R2- Memory-Augmented · 记忆增强型

06/09

User Interaction 用户交互 → R2 LLM Processor 大模型处理器 → Response 输出响应

## 四级记忆

### 一级记忆 | Sensory Buffer
当前会话 · 感知缓冲区  
**当前会话** 实时感知与临时信息 快速读写

### 二级记忆 | Short-term Memory
短期记忆 · 近期上下文  
**短期记忆** 保存近期对话与状态 支持连续交互

### 三级记忆 | Long-term Memory
长期记忆 · 稳定知识与经验  
**长期记忆** 沉淀重要事实与经验 长期可追溯

### 四级记忆 | External Vector DB
外部向量数据库 · 知识库  
**向量库** 海量知识语义检索 跨会话复用

## 让Agent真正有记忆
四级记忆 当前会话 → 短期 → 长期 → 向量库 自动搬运清理

### 优点
- 不会聊两句就失忆
- 适合长期交互

### 缺点
- 管理复杂
- 存储蹭蹭涨

R2 芯片智能体  
更强算力 · 更长记忆 · 更像人类
