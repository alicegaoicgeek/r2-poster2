# R2-5环 · 2/5

## 02 持续集成 IP集成进SoC的6种方式

六条产线覆盖 IP 从选型到集成的全链路，每一步决策都有机器可验证的确定性。

### 六条产线

1. **01 r2-reggen** — CSR 全链路一致，消灭规格漂移
2. **02 r2-c2rtl** — 算法类 IP 分钟级架构迭代与 PPA 摸底
3. **03 r2-cbbrtl** — 在流片 proven 模块上生成「敢用」的 RTL
4. **04 r2-topgen** — 像编译软件一样把多 IP 集成为一颗 SoC
5. **05 r2-buildsoc** — 像 pip 一样管理 IP 供应链与 CI/CD
6. **06 r2-ipqa** — 让选错 IP 的成本在 tapeout 之前被看见
