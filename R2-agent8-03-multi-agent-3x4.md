# R2- Multi-Agent · 多智能体协作

03/09

协调器分配 · 专科各干 · 共享知识库 · 汇总

## User Request
用户请求 / 目标 / 背景信息

## Orchestrator 协调器
- 理解目标
- 分解任务
- 分配给专科智能体
- 汇总结果

## Agent 01 · Search 智能体
信息检索专家
- 关键词理解
- 多源检索
- 筛选排序
- 输出结构化信息

## Agent 02 · Writing 智能体
内容创作专家
- 结构化写作
- 逻辑组织
- 语言润色
- 输出成稿内容

## Agent 03 · Coding 智能体
代码实现专家
- 需求理解
- 代码生成
- 运行调试
- 输出可执行代码

## Agent 04 · QA 智能体
质量保障专家
- 结果校验
- 逻辑检查
- 错误发现
- 输出修正建议

## Shared Knowledge 共享知识库
文档资料 · 案例经验 · 代码仓库 · 规范指南

## Aggregated Final Response
聚合汇总后的最终响应 / 解决方案 / 答案
