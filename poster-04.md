# 能拆能装——模块化榫头

R2 - 模块化榫头

Skill、CLI、LLM不是三个独立概念，它们是同一个榫头的三种材料。

## 01 咬合三角
Skill = 榫头（主动伸出，携带知识） · CLI = 卯口（被动容纳，提供执行接口） · LLM = 木材弹性（咬合有缝、能回弹、不折断）

## 02 IP三条判据
结构约束 = 卯口几何形状 · 反馈回路 = 缝的弹性（压力越大咬合越紧） · 组织背书 = 燕尾楔形（越压越防松）。三线俱在，Skill即IP；缺一以上，Skill即便利贴。

## 03 护城河
细胞生命周期短，基因穿越周期。真正的护城河不在Skill的仓库里，在设计和演化Harness的判断力中。

| 判据维度 | 榫卯映射 | 决定什么 |
|---|---|---|
| 结构约束 | 卯口几何形状 | 接口标准化决定可组合性 |
| 反馈回路 | 缝的弹性（压力越大越紧） | 使用强度即质量信号 |
| 组织背书 | 燕尾楔形（越压越防松） | 采纳规模即为护城河 |
