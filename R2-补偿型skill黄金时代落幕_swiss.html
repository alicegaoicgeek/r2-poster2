四问审计 + 空跑一周 · delete before add

真正审的是：谁拿走了任务控制权
每个 skill 四问
01卸了之后，质量/速度是否差不多或更好？→ 是则删/归档
02补偿规划/纪律，还是提供领域事实/工具/验收？→ 补偿型优先删
03always-on 还是用户显式调用？→ always-on 默认关
04删掉「Be thorough / ALWAYS…」后行为有变吗？→ 无变 = no-op 砍段
默认动作：先空跑一周
1备份 skill 列表
2全卸补偿型 mega-bundle
3只留 CLAUDE/AGENTS.md + memory + 手动 1～3 个
4强模型跑真实任务 3～7 天
5只把明显更好的加回来
